Chrome Extension - Google Chat Alert

🔒 1. Introdução

Esta Política de Privacidade descreve como a extensão "Google Chat Alert" coleta, usa e protege as informações quando você utiliza nossa extensão do Chrome.

📊 2. Informações que Coletamos

🚫 2.1 Dados NÃO Coletados

✅Nossa extensão NÃO coleta, armazena, transmite ou compartilha nenhum tipo de dados pessoais, incluindo:

  • Informações de identificação pessoal
  • Dados de navegação
  • Cookies
  • Histórico de navegação
  • Informações de login
  • Conteúdo de mensagens do Google Chat
  • Nomes de contatos ou conversas

💻 2.2 Funcionamento Local

A extensão opera inteiramente no seu navegador local e não envia nenhuma informação para servidores externos.

🔑 3. Permissões Utilizadas

3.1 Permissão "storage"

Utilizada exclusivamente para salvar as preferências do usuário (cor de alerta, modo piscar, regras de filtragem e tema detectado) localmente no navegador, para que persistam entre sessões.

3.2 Permissão de host (https://chat.google.com/*)

Necessária para injetar o content script e o CSS na página do Google Chat, permitindo detectar mensagens não lidas no DOM e aplicar o alerta visual (mudança de cor de fundo) na área "Início".

⚙️ 4. Funcionalidade da Extensão

A extensão adiciona alertas visuais ao Google Chat quando há mensagens não lidas:

  • Detecta mensagens não lidas via atributos do DOM (aria-label e título da página)
  • Muda a cor de fundo da área "Início" para alertar visualmente o usuário
  • Permite personalizar a cor de alerta, ativar animação pulsante e configurar regras de filtragem (Excluir/VIP)
  • Detecta automaticamente o tema (dark/light) para aplicar cores adequadas

Todas essas ações são realizadas localmente no seu navegador, sem transmissão de dados.

🌐 5. Uso de Dados de Terceiros

A extensão não utiliza:

  • Analytics ou ferramentas de rastreamento
  • Serviços de terceiros
  • APIs externas
  • Conexões de rede

💾 6. Armazenamento de Dados

A extensão armazena apenas suas preferências de configuração localmente no navegador via chrome.storage:

  • Cor de alerta escolhida (por tema)
  • Estado de ativação da extensão e do modo piscar
  • Regras de filtragem (nomes para Excluir ou VIP)
  • Tema detectado (dark/light)

Nenhum dado pessoal, mensagem ou conteúdo de conversa é armazenado.

🤝 7. Compartilhamento de Informações

Não compartilhamos nenhuma informação porque não coletamos nenhum dado pessoal. As configurações ficam salvas exclusivamente no seu navegador.

🔐 8. Segurança

A extensão é projetada com foco em privacidade:

  • Execução Local Exclusivamente local
  • Zero Transmissão Sem transmissão de dados
  • Acesso Limitado Apenas chat.google.com
  • Permissão Mínima Apenas storage
  • Open Source Código auditável

📝 9. Alterações nesta Política

Notificaremos sobre mudanças significativas nesta política através de uma atualização da extensão na Chrome Web Store.

📋 Resumo

O Google Chat Alert é uma extensão de alerta visual que funciona apenas localmente no seu navegador. Não coleta, armazena ou transmite nenhum dado pessoal — apenas salva suas preferências de configuração localmente.

🔒 1. Introduction

This Privacy Policy describes how the "Google Chat Alert" extension collects, uses, and protects information when you use our Chrome extension.

📊 2. Information We Collect

🚫 2.1 Data NOT Collected

✅Our extension does NOT collect, store, transmit, or share any type of personal data, including:

  • Personal identification information
  • Browsing data
  • Cookies
  • Browsing history
  • Login information
  • Google Chat message content
  • Contact or conversation names

💻 2.2 Local Operation

The extension operates entirely in your local browser and does not send any information to external servers.

🔑 3. Permissions Used

3.1 "storage" Permission

Used exclusively to save user preferences (alert color, blink mode, filtering rules, and detected theme) locally in the browser, so they persist between sessions.

3.2 Host Permission (https://chat.google.com/*)

Required to inject the content script and CSS into the Google Chat page, enabling the detection of unread messages in the DOM and applying the visual alert (background color change) in the "Home" area.

⚙️ 4. Extension Functionality

The extension adds visual alerts to Google Chat when there are unread messages:

  • Detects unread messages via DOM attributes (aria-label and page title)
  • Changes the background color of the "Home" area to visually alert the user
  • Allows customizing the alert color, enabling a blinking animation, and configuring filtering rules (Exclude/VIP)
  • Automatically detects the theme (dark/light) to apply appropriate colors

All these actions are performed locally in your browser, without data transmission.

🌐 5. Third-Party Data Usage

The extension does not use:

  • Analytics or tracking tools
  • Third-party services
  • External APIs
  • Network connections

💾 6. Data Storage

The extension only stores your configuration preferences locally in the browser via chrome.storage:

  • Chosen alert color (per theme)
  • Extension and blink mode activation state
  • Filtering rules (names for Exclude or VIP)
  • Detected theme (dark/light)

No personal data, messages, or conversation content is stored.

🤝 7. Information Sharing

We do not share any information because we do not collect any personal data. Settings are saved exclusively in your browser.

🔐 8. Security

The extension is designed with a focus on privacy:

  • Local Execution Exclusively local
  • Zero Transmission No data transmission
  • Limited Access Only chat.google.com
  • Minimal Permission Only storage
  • Open Source Auditable code
